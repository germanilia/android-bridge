# Mobile Second Brain Personas

## P1 — Owner-User

### Profile
- Uses a Samsung Android phone and Mac as personal devices.
- Keeps a personal Second Brain as Markdown nodes on the Mac.
- Wants phone access without turning Second Brain into a cloud/SaaS product.

### Goals
- Browse and read all Second Brain Markdown nodes from Android.
- Create, edit, and delete notes on Android while home or offline.
- Trust that changes sync back to the Mac automatically when devices reconnect.
- Search notes locally on Android.
- Avoid leaking note contents to logs, cloud services, or third parties.

### Pain Points
- Notes are currently Mac-centered.
- Offline mobile edits can be lost without a durable queue.
- Sync conflicts need predictable behavior.
- Debug info is useful, but note content must stay private.

### Story Mapping
- MSB-US-1 through MSB-US-10.
